A modern, neon-themed personal portfolio website featuring smooth animations, responsive layout, a custom typing effect, a mini frontend-only chatbot, and a glowing night-mode UI.

This portfolio showcases my skills in frontend development, UI/UX, micro-interactions, and creative web aesthetics.

Features
Neon / Night Mode UI: Dark futuristic theme, Neon cyan & neon pink accents, Soft glassmorphism cards, Animated glow effects
Dynamic Hero Section: Custom typing animation, Bold gradient heading, Smooth scroll for CTA buttons
Interactive Skills Section: Hover pop animation, Neon-gradient icons, Accessible and keyboard-friendly
Project Carousel: Sliding project cards, Minimal dot controls
Mock AI Chatbot (Frontend Only): Floating neon chat button, Expandable chatbot drawer, Dummy conversation simulation
